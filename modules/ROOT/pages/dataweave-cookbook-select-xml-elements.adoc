The XML markup language requires a single root node. To return one of multiple child elements, your script can select the value of the child element by applying a DataWeave selector to the name of the subelement or an index (such as `[1]`). The script must also use the DataWeave key syntax to specify a new element name. The following examples illustrate these options.
