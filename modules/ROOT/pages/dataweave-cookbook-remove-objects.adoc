= Remove Objects Containing Specified Key-Value Pairs

This DataWeave example removes all objects that contain a set of key-value pairs from an array of objects.
//LINK TO DW 1.0 EXAMPLES:
include::partial$dataweave1-links.adoc[tag=dataweave1Examples]


This example uses:

* `filter` to iterate over objects in the input array and return values that match the results of an expression. Each object contains a set of key-value pairs.
* `contains` function to identify the key-value pairs specified in the `dropThese` variable. This variable is an array of objects containing the key-value pairs to remove.
+
`not` negates the result of the `contains` function to make `filter` remove the key-value pairs identified in `dropThese` from the finale output. (Without `not`, the result produces an array of objects with only the undesired key-value pairs.)
